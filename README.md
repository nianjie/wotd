# wotd
#### Word of the day of english, a tiny tool for foreign language learners. 

#### Architecture
 - Backend 
   1. NodeJS(Heroku) 
   2. NoSQL DB(Firebase)
 - Frontend
   1. WeChat(Alibaba Cloud)

#### RESTful APIs
```
ROOT
  |- wotd
  |- alphabetical
  |- chronological
       |- YYYY/MM/DD
       |- today
  |- count
  |- random
```
